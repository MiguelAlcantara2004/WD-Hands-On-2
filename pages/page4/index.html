Lelouch


Lelouch vi Britannia (ルルーシュ・ヴィ・ブリタニア, Rurūshu vui Buritania) is the main protagonist of Code Geass: Lelouch of the Rebellion and Code Geass: Lelouch of the Rebellion R2 and plays a supporting role in side stories like Code Geass: Oz the Reflection and Code Geass: Lancelot & Guren.
